## 重要概念
> 資料庫是什麼?
>> 指針對某特定需求而被收集在一起的相關資料。  
>> 按照資料模型及正規化步驟加以處理,讓儲存在資料庫中的資料盡量避免重複性。  
>> 可以透過各式各樣公用程式讓使用者可以快速、有效及正確的獲得其所需要的資料。
